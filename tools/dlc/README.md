# 전장의 발큐리아 2 (NPJH50145) — DLC 한글패치

본편 한글패치의 연장으로 **DLC 22개 미션**을 한국어화했습니다.

## 무엇이 번역되었나
DLC 각 미션의 플레이어 노출 텍스트 전부:
- **미션명** (예: 『각한의곡』, 『궁극의 도전』, 『푸른종말』)
- **적 에이스명** (예: 학살자시하라, 번개의 샤카이, 철벽의 니셰오)
- **보상 무기 부품명** (예: 강화 대전차창부품A형, 연사 저격용 배럴B형)
- **미션 브리핑 / 승리조건** (전문)

합계 **22개 파일, 67개 문자열**. 용어는 본편 용어집과 일치(아반/제리/코제트, 점거/전멸/본거점, 정찰병·돌격병·대전차병·지원병·기갑병 등).

미번역: `game_info.mxe`(텍스트 없음), `omap_set/grd/omp/mlx`(맵·수치 데이터, 개발용 내부 라벨뿐, 플레이어 비노출). DL2B는 미션이 없는 공용 부품 파일이라 원본 그대로.

## 기술 요약
- DLC의 `DLxx_DATA.EDAT`는 NPDRM(EDAT) 복호화가 이미 벗겨진 **생 CPK**. 본편 `ODIN.CPK`와 달리 파일 레벨 key\*3+1 암호화가 **없고**, 2KB 프리앰블 뒤 `MXEN`(≈0x800)의 **MXEC 데이터 영역만 rolling-XOR**(flag 0x40000).
- 패치 = roll-decrypt → 한글 **동일 바이트 길이** 치환(0x00 패딩) → roll-encrypt(플래그 보존) → CPK **제자리 덮어쓰기**. 파일 크기·TOC·비미션 바이트 전부 불변.
- 한글은 본편 배포 폰트 매핑(`tl/mapping_v17.json`, 1224 음절)으로 인코딩 → 사용 음절 전부 커버(미매핑 0). 각 한글 음절 = SJIS 한자 슬롯 2바이트.

## ⚠️ 사용 조건
DLC엔 폰트가 없어 **본편 패치의 한글 폰트(`ODIN_FONT_16.BF1`)를 공유**합니다.
- **반드시 한글패치된 본편(v25 텍스트판 이상)과 함께 사용**하세요. 원본 일본판과 함께 쓰면 글자가 깨진 한자로 보입니다.
- 출력물은 입력과 **동일 포맷(생 CPK)**이므로, 원본 DLC가 동작하던 환경(PPSSPP 등 복호화 DLC 사용 환경)에서 그대로 동작합니다.

## 설치
`out/NPJH50145/` 전체를 원래 DLC 위치(`.../DLC/NPJH50145/`)에 **덮어쓰기**.
- 패치된 22개 `DLxx_DATA.EDAT` + 원본 그대로인 `DLxx.EDAT`/`PARAM.PBP`/`DL2B`.
- 만일을 위해 원본 폴더는 백업 권장.

## 파일
- `out/NPJH50145/` — 배포용 드롭인 세트 (원본과 크기·구조 동일)
- `dlc_translation_manifest.json` — JP↔KO 대조표 (67쌍, 재편집용)
- `dlc_lib.py` / `build_dlc.py` / `dlc_ko.py` — 추출·주입 도구 + 번역 원본
